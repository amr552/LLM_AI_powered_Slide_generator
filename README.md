# SlidePrompt UX Builder (LLM-Powered)

A lightweight AI application that helps users produce **meaningful, structured prompts** for building presentation slides.

This project focuses on the “last-mile” UX problem: most users know the topic, but struggle to write a clear prompt that yields a strong slide deck. SlidePrompt solves that with two modes:

1) **Guided Prompt Mode (User Content)**
   - The app guides the user to write a high-quality prompt and slide content using structured inputs and a prompt coach UI.

2) **AI Slide Content Mode (LLM Content)**
   - The user provides a topic + constraints and the app generates a concise, slide-by-slide outline using an LLM (OpenAI API).

The output is intentionally **short and structured** to keep cost low and results consistent.

---

## Key Features

- Two workflow modes:
  - **User-written with UX coaching**
  - **AI-generated slide content**
- Strong prompt UX:
  - Audience, goal, tone, duration, slide count, level (beginner/advanced)
  - Must-include / must-avoid constraints
  - “Good prompt” live preview
- Token-efficient generation:
  - Short format, controlled max_tokens
  - Optional “Ultra-short” mode
- Predictable slide structure:
  - Title + 3–5 bullets + speaker notes (optional)
- Production-grade error handling:
  - Invalid API keys, quota errors, network issues
- Dual UI:
  - **Streamlit** (product UX)
  - **Gradio** (demo & share)

---

## Tech Stack

- Python 3.10+
- OpenAI API
- Streamlit / Gradio
- python-dotenv
